Encryption Algorithms Documentation

Caesar Cipher

A simple substitution cipher that shifts each letter in the plaintext by a fixed number of positions in the alphabet.

How It Works

Using a numerical key (shift value), each letter is replaced by a letter some fixed number of positions down the alphabet.

Use Cases

  • Basic text encryption for learning purposes
  • Teaching fundamentals of cryptography

Playfair Cipher

A digraph substitution cipher that encrypts pairs of letters instead of single letters.

How It Works

Uses a 5x5 grid of letters constructed using a keyword. Letters are encrypted in pairs according to their positions in the grid.

Use Cases

  • Historical cryptography studies
  • More secure alternative to simple substitution

Vigenère Cipher

A polyalphabetic substitution cipher using a keyword to create multiple shift patterns.

How It Works

Uses a keyword to determine shift values for each letter in the plaintext, creating multiple Caesar ciphers used in sequence.

Use Cases

  • More complex encryption for learning
  • Understanding polyalphabetic substitution

RSA Encryption

A public-key cryptosystem that uses paired keys for encryption and decryption.

How It Works

Uses two different keys: a public key for encryption and a private key for decryption. Based on the mathematical properties of large prime numbers.

Use Cases

  • Secure data transmission
  • Digital signatures
